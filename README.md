# storyforge-app-platform

A small StoryForge UI intended for **DigitalOcean App Platform**.

It talks to a VPC-only gateway (`tinybox-compute-node-gateway`) which in turn talks to Tinybox over Tailscale.

## Status

Deprecated: replaced by https://github.com/I0Q/storyforge (main app) and https://github.com/I0Q/storyforge-infra (infra/spec notes).
This repo was an early App Platform proof-of-concept UI.
